# Backend-сервіс інтернет-магазину

Цей проєкт є курсовою роботою з дисципліни **«Об’єктно-орієнтоване програмування»**.

Метою роботи є розробка серверної частини інтернет-магазину з використанням Java та Spring Boot. Застосунок надає REST API для роботи з товарами, категоріями, клієнтами, замовленнями та позиціями замовлень.

## Основні можливості

- керування товарами;
- керування категоріями;
- керування клієнтами;
- створення та перегляд замовлень;
- робота з позиціями замовлення;
- пошук даних;
- отримання простої аналітики;
- валідація вхідних даних;
- централізована обробка помилок;
- документація API через Swagger UI.

## Основні сутності

- `Product` — товар;
- `Category` — категорія;
- `Customer` — клієнт;
- `Order` — замовлення;
- `OrderItem` — позиція замовлення;
- `OrderStatus` — статус замовлення.
